# Mercado University V2

Premium Spanish education SaaS for practical online income skills.

## Launch courses

- Trading desde Cero
- Shopify desde Cero
- TikTok Shop
- UGC Creator
- Digital Agency
- Marketing
- Sales

AI for Business is intentionally excluded from the first development phase.

## Architecture

Every course uses the same reusable engine:

Course → Module → Lesson → YouTube Video → Summary → Action Steps →
Quiz → Flashcards → Notes → Assignment → Resources → Certificate

## Development order

1. Design system
2. Platform architecture
3. Shared course engine
4. Authentication and subscriptions
5. Progress and learning tools
6. Curated course content
7. Testing
8. Deployment
